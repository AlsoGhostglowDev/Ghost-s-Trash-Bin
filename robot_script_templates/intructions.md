![image](https://github.com/user-attachments/assets/4be99589-d6e4-4909-b6c7-d02fcea57457)

## `autofarm.lua`

- Firstly, place the robot on the corner, it will setup the farm for you (if you haven't yet) <br>
![image](https://github.com/user-attachments/assets/22cb7279-5c7b-4866-abdf-390c3f4a0dd6)

- Add seeds to your robot's first slot. The robot will auto-farm and collect the drops as it farms <br>
![image](https://github.com/user-attachments/assets/da614807-c6e4-4f13-9958-35c8d3bb650b)

- The robot will ocassionally sleep after there was no crops to farm (idle for a period of time) <br>
The sleep duration can be changed with the restDur variable

- If you've nothing to change in the file, then use: <br>
`wget https://raw.githubusercontent.com/AlsoGhostglowDev/Ghost-s-Trash-Bin/main/robot_script_templates/autofarm.lua` <br><br>
Else, copy the script's code and paste it in a new lua file with the <kbd>INSERT</kbd> key
